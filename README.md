# IllitLevels

IllitLevels — плагин уровней и опыта (1–100) для Paper/Spigot 1.20.1.  
Уровень прокачивается за опыт, который можно выдавать за квесты.  
Требуемый опыт на уровень растёт по сложной формуле, а последние 10 уровней прокачиваются очень тяжело.

---

## Команды

### Игроки
(Игроки обычно используют уровни через квесты/награды. Отдельных обязательных команд для игроков нет.)

### Админ-команды (требуется `illit.levels`)
Все админ-команды выполняются через `illit`:

**Уровни**
- `/illit lvl give <ник> <кол-во>` — добавить уровни
- `/illit lvl remove <ник> <кол-во>` — снять уровни (ниже 1 уровень не опускается)
- `/illit lvl set <ник> <уровень>` — установить уровень

**Опыт**
- `/illit exp give <ник> <кол-во>` — добавить опыт
- `/illit exp remove <ник> <кол-во>` — снять опыт (уровень уменьшится, если опыта стало меньше нормы)
- `/illit exp set <ник> <кол-во>` — установить опыт на текущем уровне

**Топ**
- `/illit top` — топ по уровню

## PlaceholderAPI плейсхолдеры

### Основные
- `%illit_level%` — текущий уровень
- `%illit_exp%` — текущий опыт на уровне
- `%illit_level_next%` — следующий уровень
- `%illit_exp_next%` — сколько опыта нужно до следующего уровня

### Форматирование/прогресс
- `%illit_progress%` — прогресс вида `текущее/нужно`
- `%illit_progress_percent%` — прогресс в процентах
- `%illit_progress_bar%` — прогресс-бар

### Плейсхолдеры топа (1–10)
- `%illit_top_name_1%` … `%illit_top_name_10%` — ник в топе
- `%illit_top_level_1%` … `%illit_top_level_10%` — уровень в топе

## Админ-команды и пермишены

Админ-доступ ко всем админ-командам:
- `illit.levels`

Пример выдачи (LuckPerms):
- `lp group admin permission set illit.levels true`
